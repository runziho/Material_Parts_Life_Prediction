# Material_Parts_Life_Prediction 
- **소재부품 수명 예측 및 설계 지원 AI 시스템 개발**

---

## 👥 팀명 : 올클 
**팀원 : 강다영, 이혜원, 최지호**

---

## 📌 Project Overview
고온·고압 환경에서 사용되는 **내열강 및 초내열합금 소재 부품의 수명을 예측하는 인공지능 기반 시스템을 개발하는 프로젝트**입니다.  

합금 성분, 온도 등의 입력 데이터를 기반으로 **크립 수명(Creep Life)**을 예측하며, 단순 예측을 넘어  
사용자가 직접 성분을 조절하고 수명 변화를 확인할 수 있는 **인터랙티브 시뮬레이션 기능**을 포함합니다.  

또한, 비전문 사용자도 쉽게 이해할 수 있도록 **AI 기반 설명 및 결과 해석 기능**을 제공하여  
실제 활용 가능한 형태의 서비스 구현을 목표로 합니다.

---

## ⚙️ Tech Stack
- Python  
- TensorFlow / Keras (ANN 기반 앙상블 모델)  
- Scikit-learn  
- Pandas / NumPy  
- Matplotlib  
- Streamlit (웹 UI)  

---

## 🎯 Objectives
- 합금 성분 및 환경 데이터 기반 **크립 수명 예측 모델 개발**
- ANN 기반 **앙상블(Committee) 모델을 통한 예측 성능 향상**
- 로그 스케일 수명 값을 실제 시간 단위로 변환하여 사용자 이해도 개선
- 사용자가 합금 성분을 직접 조절할 수 있는 **시뮬레이션 기능 구현**
- 예측 결과 및 변화량을 직관적으로 보여주는 **시각화 시스템 구축**
- 비전문가도 사용할 수 있도록 **AI 기반 질의응답 및 결과 해석 기능 제공**

---

## 🚀 Key Features

### 🔹 1. 수명 예측 기능
- 입력된 합금 성분 데이터를 기반으로 크립 수명 예측
- ANN 기반 다중 모델 앙상블 평균값 활용

---

### 🔹 2. 실제 시간 변환
- log10 기반 예측값을 실제 시간(hour) 단위로 변환  
- 사용자 관점에서 직관적인 결과 제공  

---

### 🔹 3. 합금 성분 시뮬레이션 (핵심 기능)
- 특정 샘플 선택 후 주요 성분(Cr, Ni 등) 직접 조절 가능
- 성분 변화에 따른 수명 변화 실시간 반영

---

### 🔹 4. 수명 변화 비교 시스템
- 기존 수명 vs 변경 수명 비교
- 변화량 및 변화율 자동 계산
- 시각적 비교 그래프 제공

---

### 🔹 5. 인터랙티브 시각화
- 선 그래프 기반 수명 변화 표현
- 미세 변화도 확인 가능하도록 축 범위 자동 조정

---

### 🔹 6. AI Assistant (기초 지식 지원)
- 크립, 합금, 온도 등 핵심 개념 설명
- 비전문 사용자도 쉽게 이해 가능

---

## 📊 Expected Outcome
- 소재 부품 수명 예측 AI 모델
- 사용자 인터랙션 기반 시뮬레이션 시스템
- 데이터 기반 의사결정 지원 도구
- 웹 기반 AI 서비스 형태의 결과물
