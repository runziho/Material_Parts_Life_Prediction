# Material_Parts_Life_Prediction 
- **소재부품 수명 예측 모델 개발**

## 팀명 : 올클 (임시)
**팀원 : 강다영, 이혜원, 최지호**

## 📌 Project Overview
고온·고압 환경에서 사용되는 **내열강 및 초내열합금 소재 부품의 수명을 예측하는 인공지능 모델을 개발하는 프로젝트**입니다.  
합금 성분, 온도 조건 등의 데이터를 활용하여 **크립 수명(Creep Life)과 강도를 예측**하고, 부품의 유지보수 및 신뢰성 향상에 기여하는 것을 목표로 합니다.

## 🏢 Project Info
- **기업**: SK 이노베이션  
- **프로젝트 유형**: 산학 협력 AI/SW 프로젝트  

## ⚙️ Tech Stack
- Python
- Machine Learning / AI
- Data Analysis
- GUI Programming

## 🎯 Objectives
- 합금 성분 데이터 기반 **크립 수명 및 강도 데이터 분석**
- AI 기반 **소재 부품 수명 예측 모델 개발**
- 입력 변수 중요도 및 신뢰도 분석
- 실험 데이터와 예측 모델 성능 비교
- GUI 기반 **모델 학습 및 평가 프로그램 개발**

## 📊 Expected Outcome
- 소재 부품 수명 예측 AI 모델
- 데이터 분석 결과
- 모델 학습 및 평가 GUI 프로그램
